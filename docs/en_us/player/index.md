# Player Guide

This section of the documentation is specifically for players and server administrators, aiming to help them learn how to install, use Git Parcel, and troubleshoot.

All available guides are listed in the sidebar.

If you encounter any issues, please report them on [GitHub](https://github.com/Git-Parcel/Mod/issues).

> [!WARNING]
>
> Unofficial versions of the mod do not guarantee stable functionality, and data saved in them may not be loaded correctly in future versions. Use with caution.

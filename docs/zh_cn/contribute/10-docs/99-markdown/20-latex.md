## 符号

### 逻辑符号

| 中文名称               | LaTeX                           | 常见用法示例          | 备注                                            |
| ---------------------- | ------------------------------- | --------------------- | ----------------------------------------------- |
| **与** (AND)           | `\land`                         | $A \land B$           | 最常用，表示逻辑合取。也可用 `&` 连接文本。     |
| **或** (OR)            | `\lor`                          | $A \lor B$            | 表示逻辑析取。也可用 `+` 连接文本（如 $A+B$）。 |
| **非** (NOT)           | `\neg`                          | $\neg A$              | 否定符号，通常放在变量前。                      |
| **蕴含** (IMPLIES)     | `\implies`                      | $A \implies B$        | 表示蕴涵关系。也可用 `->` 或 `>`。              |
| **等价** (EQUIV)       | `\iff`                          | $A \iff B$            | 双向蕴含，表示充要条件。也可用 `<->`。          |
| **存在** (EXISTS)      | `\exists`                       | $\exists x, P(x)$     | 量词符号，用于存在性命题。                      |
| **全称** (ALL)         | `\forall`                       | $\forall x, P(x)$     | 量词符号，表示对任意元素成立。                  |
| **有限域** (FIELD)     | `F` / `\mathbb{F}`              | $F$ 或 $\mathbb{F}_p$ | 常用来表示数域（如整数模 $p$）。                |
| **布尔代数** (BOOLEAN) | `B` / `\mathcal{B}`             | $B$ 或 $\mathcal{B}$  | 有时用于表示布尔环/代数结构。                   |
| **集合论符号**         | `\in`, `\notin`, `\cup`, `\cap` | $x \in S$, $A \cup B$ | 专门用于集合运算和元素归属关系。                |
| **逻辑连接词 (文本)**  | `and`, `or`, `not`              | $A \text{ and } B$    | 在数学证明中常用，避免歧义（如与乘法区分）。    |

### 代数与函数运算符号

| 中文名称                            | 英文全称 (LaTeX)                    | 常见用法示例                                   | 备注                                                                  |
| :---------------------------------- | :---------------------------------- | :--------------------------------------------- | :-------------------------------------------------------------------- |
| **加** (Sum/Plus)                   | `+` / `\oplus`                      | $a + b$ 或 $\bigoplus_{i=1}^n x_i$             | 普通加法用 `+`；环论、向量空间中的逐点运算常用 `\oplus`。             |
| **减** (Difference/Minus)           | `-` / `\ominus`                     | $a - b$ 或 $\ominus$                           | 普通减法用 `-`；群论中有时用 `\ominus` 表示差集或特定运算。           |
| **乘** (Product/Multiplication)     | `\cdot` / `\times` / `$\prod$`      | $a \cdot b$, $x \times y$, $\prod_{i=1}^n a_i$ | `·` 表示乘法；`\times` 常用在函数图像或坐标系中；`\prod` 表示连乘积。 |
| **除** (Division/Quotient)          | `\div` / `$\frac{a}{b}$`            | $a \div b$, $\frac{a}{b}$, $a/b$               | LaTeX 中通常显式使用分数形式 `\frac`，`\div` 用于文本流。             |
| **幂** (Power/Exponentiation)       | `^`, `$\uparrow$`, `$\circledcirc$` | $a^n$, $x \uparrow y$, $\lambda^n$             | 指数通常用 `^`；佩尔符号 `\uparrow` 表示幂运算（如康托尔序数）。      |
| **根** (Root)                       | `\sqrt`                             | $\sqrt{x}$                                     | 方根符号                                                              |
| **模运算** (Modulo)                 | `\mod` / `\equiv`                   | $a \pmod n$, $a \equiv b \pmod n$              | `\pmod` 表示取余；`\equiv` 通常表示同余关系（数论）或恒等式。         |
| **函数复合** (Function Composition) | `\circ`                             | $(f \cdot g)(x)$, $f(g(x))$, $f \circ g$       | 点号 `·` 常用作乘积                                                   |
| **逆运算** (Inverse)                | `$^{-1}$`, `\inverse`               | $A^{-1}$, $f^{-1}(x)$                          | 表示矩阵、数或函数的逆变换/逆映射。                                   |

### 集合与关系符号

| 中文名称                        | 英文全称 (LaTeX)                    | 常见用法示例                      | 备注                                |
| :------------------------------ | :---------------------------------- | :-------------------------------- | :---------------------------------- |
| **包含** (Subset)               | `$\subseteq$` / `\sqsubseteq`       | $A \subseteq B$                   | 表示子集关系（允许相等）。          |
| **真包含** (Proper Subset)      | `$\subsetneq$`, `$\subset` (旧)`    | $A \subsetneq B$                  | 表示真子集（不相等）。              |
| **等价类** (Equivalence Class)  | `[\cdot]`, `\{\cdot\}`              | $[x]_\sim$, $\{x, y\}$            | 括号内加下标表示等价关系下的类。    |
| **开集/闭集** (Open/Closed Set) | `$\text{Int}(A)$`, `$\text{Cl}(A)$` | $U \in \mathcal{T}$, $\partial A$ | 拓扑学中常用，$\partial$ 表示边界。 |
| **点属于集合** (Point in Set)   | `$\bullet$`, `$\odot$`              | $x \in S$, $\bullet \in D$        | 有时用于几何图形中表示点在区域内。  |

### 不等式与极限符号

| 中文名称                          | 英文全称 (LaTeX)                   | 常见用法示例                             | 备注                                                                      |
| :-------------------------------- | :--------------------------------- | :--------------------------------------- | :------------------------------------------------------------------------ |
| **小于/大于** (Less/Greater Than) | `<`, `>`, `\gtrless`, `\lesseqgtr` | $x < y$, $a \ge b$, $\alpha > 0.5$       | `\leq` / `\geq` 表示带等号的不等式。                                      |
| **趋近/极限** (Approach/Limit)    | `$\to$`, `$\sim$`, `$\asymp$`      | $x \to 0$, $a_n \to L$, $f(x) \sim g(x)$ | `\to` 表示趋向；`\sim` 表示渐近等价（如小 o, p）；`\asymp` 表示渐近同阶。 |
| **序关系** (Ordering)             | `$\preceq$`, `$\succcurlyeq$`      | $x \succeq y$, $\alpha \preceq \beta$    | 用于偏序集或优化问题中的目标函数比较。                                    |

### 量词与逻辑组合的变体

| 中文名称                 | 英文全称 (LaTeX)               | 常见用法示例                              | 备注                                                                    |
| :----------------------- | :----------------------------- | :---------------------------------------- | :---------------------------------------------------------------------- |
| **存在与** (Exists And)  | `$\exists x, P(x)$ and $Q(x)$` | $\exists x \text{ s.t. } P(x) \land Q(x)$ | 有时写作 `\exists x: P(x) \wedge Q(x)`。                                |
| **全称或** (All Or)      | `$\forall x, P(x)$ or $Q(x)$`  | $\forall x \in S: P(x)$                   | 在定义域约束下使用冒号 `:` 替代逗号更常见，如 `\forall x \in X, P(x)`。 |
| **蕴含与** (Implies And) | `$A \implies B \land C$`       | $P \to Q$, $(p \wedge q) \to r$           | 注意括号的使用避免歧义。                                                |
